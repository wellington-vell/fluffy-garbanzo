## Pré-requisitos

- Node.js **22+**
- Pnpm
- Ambiente configurado para rodar aplicações React Native (Android Studio / Emulador ou dispositivo físico com USB Debug habilitado)

---

## Instalação

### Instalação das dependências

1. Certifique-se de ter o [pnpm instalado](https://pnpm.io/pt/installation).
2. No diretório do projeto, execute:

```bash
pnpm install
```

## Rodando aplicação

faça prebuild da aplicação

```bash
npx expo prebuild
```

roda a aplicação

```bash
pnpm run android
```
